#  Student Management System (React Router Edition)

This is a React-based Student Management System built to practice and implement core React concepts such as React Router, CRUD operations, authentication logic, and local storage.

---

##  Features

-  Login & Logout System
-  Protected Routes (Authentication-based navigation)
-  Add Student Record
-  Display Student Records
-  Search Student Records
-  Update Student Details
-  Delete Student Records
-  Dynamic Routing using **:id**
-  Data persistence using LocalStorage
-  404 Page Not Found handling
-  React Router Navigation (Link, Navigate, useNavigate)

---

##  Tech Stack

- React JS
- React Router DOM
- JavaScript (ES6+)
- CSS3
- LocalStorage

##  Routing Overview

/login              → Login Page
/system             → Dashboard Home
/showStudents       → Display Students
/addStudent         → Add Student
/searchStudent      → Search Student
/updateStudent/:id  → Update Student
*                   → 404 Page
